HTML Tutoring

-Welcome to HTML Tutoring!

HTML, or Hyper Text Markup Language, is a markup language used to determine the structure of a web page. It is comprised of many elements that lay out the basic components of a web page. When used in tandem with CSS, and possibly other web programming languages, it can create a cohesive, well designed web site.

-Tags

HTML is a language based on tags. Tags are elements of a page, visible or not, that help to build the structure of a web page. A tag is comprised of a < symbol, the name of the element, and a > symbol, looking like <element>. All tags must be closed, at some point as well, which is done using an almost identical tag, however with a / included in the closing tag, looking like </element>.

-Basic Form of an HTML Document

  --The Declaration Statement & HTML tag

Any HTML document begins with a declaration statement, a statement declaring the type of document. A declaration statement is a special type of tag that does not need to be closed.In an HTML page, the declaration statement looks like this:

Declaration Statement & HTML Tag Example:

Code:
<!DOCTYPE HTML>
<html>

  --The Head Section

The <head> tag indicates the head element, which is an element used to hold important, but mostly non-directly-visible information about a web page. The four major types of elements usually placed in a head section are the <title>, <meta>, <style>, and <script> elements.

-The <title> element dictates the title of the web page, which will be displayed in the name of the tab, or on the title bar of the browser.

-The <meta> element dictates important, yet invisible things about the web site, like the character set to name one example.

-The <style> element dictates a CSS stylesheet, which will be talked about more on the CSS Tutoring page. One could also link an external CSS file using the <link > element.

-The <script> element dictates JavaScript code. JavaScript is a language used to make a web page more interactive. You will learn more about this in the future.

Head Section Example (Not including <style>, <link>, or <script>:

Code:
Display:
<head>
    <title> HTML Tutoring</title>
    <meta charset="UTF-8">
</head>

-The Body Section

The <body> element of an HTML document contains all parts of the web page visible to the viewer. Here are some tags that allow for the web page to have structure.

  --Headings & Paragraph Text

In HTML, headings are different sized sub-titles for a page. They come in 6 sizes, 1 being the largest, and 6 being the smallest. The heading tag's syntax is as such: <h(x)>, where (x) is the size desired.
-In comparison, paragraph text is implemented using the <p> tag. Paragraph text is the default sized text.

Headings Example:

Code:
Display:
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
<p>Normal text looks like this.</p>

  --<span> and <div>

In HTML, pages can be divided up in order to make styling much easier. To do this, we use the <span> and <div>elements. While we will talk more about this in CSS Tutoring, you are able to name these <span>s and <div>s using the id property. This can be helpful, because as we will talk about later, you are able to link directly to specific ids in a web page. <span>s and <div>s act almost exactly the same, however a <div> will insert a new line before and after it, while a <span> will not.

Example of the <span> and <div>:

Code:
Display:
<span>Hey! </span>
<span>Hello! </span>
<div>Hi! </div>
<span id="namedSpan">
    Hi, my name is namedSpan!
</span>
Hey! Hello!
Hi!
Hi, my name is namedSpan!

  --Lists, <ol> and <ul>

Another important element of web pages are lists, which HTML allows for both ordered and unordered versions. Both types of lists are begun by either an opening <ol>, for an ordered list, or <ul> for an unordered list. Then, all list items are surrounded by a set <li> and </li> tags. Finally, the list is ended with a </ol> or </ul> tag.

Example of <ol> and <ul>:

Code:
Display:
Best Three Numbers:
<ol>
    <li>6</li>
    <li>4</li>
    <li>9</li>

Honorable Mentions:
</ol>
<ul>
    <li>2</li>
    <li>7</li>
    <li>0</li>
</ul>
Best Three Numbers:
  1. 6
  2. 4
  3. 9
Honorable Mentions:
  • 2
  • 7
  • 0

  --Links

In HTML, pages and files can be attached using hyperlinks, or links. Pages and files can be linked using the anchor tag, <a>. Tags can have certain properties, and in the case of the anchor tag, the Hypertext REFerence, or href property determines the link or file that the anchor tag links to. The text or other HTML element to act as the clickable link is placed within the anchor tag. The syntax of the anchor tag is as such:
  <a href="https://www.example.com"> Link Text </a>
You can also use <a> tags to link to direct places within web pages, using the id name you provide to each item in place of a url, which looks like this:
  <a href="#idName"> Link Text </a>

Example of the Anchor Tag:

Code:
Display:
<a href="https://www.chaseurban.me">
    Click here to go to my website!
</a>
<a href="#namedSpan">
    Click to go to the named &lt;span&gt;.
</a>
Click here to go to my website! Click to go to the named <span>

  --Images

In HTML, images can be inserted into web sites using the <img> tag. Much like the anchor tag, the image tag needs set properties to function properly. The two most important properties to set are the Source, or src; and the Alternate Text Property, or alt.
-The src is the location the picture can be located at, be it an online address, or a local relative address.
-The alt is the text that will appear when the image is unable to load.

Example:

Code:
Display:
<img src="./logo.png" alt="HTML 5 Logo"/>
<img src="bad.website.com" alt="oops"/>
