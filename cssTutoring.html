CSS Tutoring


-Welcome to CSS Tutoring!

CSS, or Cascading Style Sheets is a language used in conjunction with HTML to create well designed web sites. While HTML determines the structure and layout of a web page, CSS determines the style, and mastering CSS will enable you to create beautiful web sites.

-Selectors and Properties

CSS relies on the idea of selectors and properties. A selector is a category of HTML code to be stylized in a certain way. Properties are the values you set to style the elements of certain selectors.

  --Types of Selectors

--There are multiple types of selectors, which allow a more fine-tuned styling of a web site:
-The highest level of selector is the HTML selector. The HTML selector is used to define styles that will apply to all instances of a specific HTML tag. It is declared using only the name of the HTML element.
-Next is the class selector. The class selector is used to define a class of elements, which can be used to style any one or multiple html tags. It is declared using a period (.), along with the name of the desired class.
-Next is the id selector, which is used to define an id, for use on only one element. You may recall id from internal linking, as an id can only be used on one element so that internal linking may work properly. It is declared using a pound sign (#).
-The syntax of all CSS statements is selector{property:value;}.

Selectors Example:

Code:
<!--HTML Selector-->
body{
    property: value;
}

<!--class Selector-->
.className{
    property: value;
}

<!--id Selector-->
#idName{
    property: value;
}

  --Properties

In CSS, a selector is modified by adding properties. Every property has a set of one or more values.

Example of Font Properties:

font-family:
Arial,  Impact,  Tahoma,  Verdana...
font-style:
normal,  italic,  oblique...
font-size:
20px,  20pt,  0.25in...

Example of Text Properties:

line-height:
normal, number, percentage...
text-decoration:
underline, overline, line-though...
text-align:
left, right, center, justify
text-indent:
length, percentage
white-space:
normal, pre

  --Advanced Selectors

-Additionally, you are able togroup selectors of any type, using commas (,) in between each selector. This may allow you to apply a base style across many different selectors.
-Also, you are able to use what are called context-dependent selectors, selectors that are only used when a certain number of tags are nested inside of each other. Context-dependent selectors are declared by using multiple selectors with spaces in between.
-Also, within CSS there exist psuedo-classes of selectors. A psuedo-class is a special type of selector that is only used when a specific condition is met. For example, a style that activates on hover. A psuedo-class is declared with a colon (:) between the selector and the psuedo-class identifier.

Example:

Code:
<!--Group Selector-->
h1, h2, h3, h4, h5, h6{
    text-align: center;
}

<!--Context-Dependent Selector-->
b i{
    font-size: 16pt;
}

<!--Psuedo-Class-->
a:hover{
    font-family: Impact;
}

-Where CSS code is placed

CSS code can be implemented in one of three ways:

  --Inline CSS

Inline CSS is inserted directly into the tag of the element you wish to style. It is good for when this is the only instance of an element you wish to use across an entire web site. To implement, you use the 'style' variable in the tag declaration, as such:

Inline CSS Example:

Code:
<p style="property: value;">text</p>

  --Embedded CSS

Embedded CSS is inserted directly into the head section of the HTML document. It is useful for when a style needs to be used across one web page, but not across other pages within the web site. It is inserted into a <style> element, in the <head> element, like this:

Embedded CSS Example:

Code:
<head>
  <style>
    selector{
        property:value;
      }
  </style>
</head>

  --External CSS

External CSS is inserted into an external .css file, as the name 'external css' would imply. It is useful for when you need to reuse a style or set of classes and ids across multiple web pages on a web site. It si linked using a <link> element, like this:

External CSS Example:

Code:
---------------index.html---------------
<link rel="stylesheet" type="text/css" href="./main.css"/>

---------------main.css---------------
selector{
  property:value;
}
